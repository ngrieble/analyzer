What is Stalker Stats?

This project is for viewing interesting analytics based on Facebook interactions. User's log into their Facebook accounts which then triggers graphs to be displayed showing information ranging from who tags them in the most photos to who likes their status' the most.

Instructions

When first navigating to the site users will be prompted by a button to log into their Facebook account. Once clicking this button a pop up will show the necessary Facebook permissions for the application. If the user chooses to decline these permissions they will not be able to use the Stalker Stats app. Once a user grants permission a loading message will appear while data is pulled and graphs are generated.
